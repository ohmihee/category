# 값(value)과 리터럴(literal)
        * 리터럴 -> 소스코드의 고정된 값을 의미
    ex) 1    -> 정수              (number)
        1.1  -> 부동소수점 실수   (number)  
        "hello" or 'hello' -> 문자열    (string)
        true or false      -> 참 또는 거짓의 진리값  (boolean)
        null        -> 비어있음, 존재하지 않음       (object)
        undefined   -> 정의되어 있지 않음            (undefined)

* 변수
- let 
    -> 값 재정의 가능
- const 
    -> 값에 대한 정의는 처음 한 번만 가능, 나중에 재정의 불가.
